# Endeavour Web 2021

## _'Let\'s go'_

### Installation

- clone the repository
- use npm i
- cd client
- npm i

### Running

- Backend\
  npm start
- Front End\
  cd client\
  npm start

### Backend .env file

```
PORT = 
DB = 
GMAIL_USER = 
GMAIL_PASS = 
EMAIL_SECRET = 
SECRET = 
SECRET_2 = 
DOMAIN = 
RAZORPAY_SECRET =
RAZORPAY_KEY_ID = 
BACKDOMAIN = 
GMAIL_USER_MASS = 
GMAIL_PASS_MASS = 
```

### Frontend .env file
```
REACT_APP_BACKEND = 
REACT_APP_KEY = 
REACT_APP_FIREBASE_DOMAIN =  
REACT_APP_FIREBASE_DATABASE = 
REACT_APP_FIREBASE_PROJECT_ID =

```
